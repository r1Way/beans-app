## 🎨 这是个啥项目？

一个 Vite + React + TypeScript + Tailwind CSS 的纯前端小玩具：
**拼豆工坊**——在网页上点点点，拼出可爱像素画，还能“熨烫定型”、导出 PNG。

核心思想：**简洁、可爱、手残党友好**。

---

## ✅ 已经做了啥

### 部署上线
- 配置 GitHub Actions 自动构建 + 部署到 GitHub Pages
- 修复 CI 构建报错：
  - TypeScript 5.9 废弃了 `baseUrl`，改用 `paths`
  - 移除了仅本地可用的 `kimi-plugin-inspect-react`
  - 修复 `package-lock.json` 里无法访问的 `npm.mirrors.msh.team` 镜像地址

### 颜色系统（4 种模式）
1. **经典**：固定 14 色拼豆色板
2. **随机**：把 14 色打乱顺序显示，可重新洗牌
3. **高级**：用系统调色板选任意颜色，自动匹配到最近的拼豆色，保留最近使用的 6 个
4. **图片**：上传图片，提取图中主要颜色并映射到拼豆色板

### 豆子渲染
- 去掉了多余的渐变、高光、孔内壁阴影
- 现在是简洁的：单色圆环 + 细描边 + 通透中心孔

### 画板尺寸
- 小 16×16
- 中 24×24
- 大 32×32
- 超大 64×64

### 导出功能
- 点击“导出”会弹出设置窗口
- 可选项：
  - **透明背景**：关掉奶油色底，导出 PNG 透明底
  - **水印边框**：在图片外侧加边框，带：
    - 网页 favicon 风格的红色拼豆图标
    - 网站地址 `r1way.github.io/beans-app`
    - 当前日期
    - 不会遮挡用户作品

### 移动端优化
- “熨烫”“导出”按钮在手机上也会显示文字（不再只有图标）
- 工具按钮显示快捷键提示：
  - B = 画笔
  - E = 橡皮
  - F = 填充
  - I = 取色
  - 直接按字母键即可，不用按 Ctrl

### 交互调整
- 去掉了“双指轻点擦除”，避免误触
  - 擦除现在只能通过：橡皮工具、右键/长按（桌面）、撤销、清空
- 添加了 favicon（红色拼豆圆环）
